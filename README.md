# Imagofono

Son 3 elementos: 
HandPoseV2
Código P5JS que interpreta los movimientos de la mano y envía las señales via OSC

P5-OSC-Bridge
Utilizando Node, envía la recibida por HandPose a Sonic PI

HandPoseV2 - SonicPi
Son todos los códigos de SonicPi.
Se recomienda que se ejecuten todo al mismo tiempo
